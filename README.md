# AI-Enabled-IoT-ECG-Monitoring-kit-with-Real-Time-Blynk-Visualization-for-drone
An AI-enabled IoT ECG monitoring system using ESP32 that captures real-time ECG signals, displays continuous waveforms, calculates heart rate, and analyzes rhythm using AI. Data is visualized locally and remotely via Blynk for smart healthcare monitoring.This system can help save the lives of heart patients in emergency situations.
